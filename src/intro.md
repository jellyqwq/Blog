# 关于我

你好，我是黄国栋，也常用 Jelly 这个名字。

我本科专业是大数据工程技术，平时主要写 Python、Web 页面和一些数据处理相关的小工具。现在我正在把自己的方向更集中地转到前端开发上，目标是做能真正服务业务的 Web 应用：中后台系统、数据看板、可视化页面、接口联调和工程化落地。

我喜欢把技术用在具体问题上。以前做过课表推送、小说和漫画采集、API 聚合服务，也参加过人工智能创意赛、数学建模和程序设计竞赛。这些经历让我比较熟悉“数据从哪里来、怎么处理、怎么展示给用户”这一整条链路。

## 当前方向

- 前端开发：Vue3、JavaScript、TypeScript、Element Plus、ECharts
- 后端与数据处理：Python、Flask、MySQL、MongoDB、Pandas
- 工程化与部署：Git、Docker、Linux、Nginx

我目前希望找一份广州的前端开发工作，尤其关注中后台系统、数据可视化、业务工具和前后端联调相关方向。
